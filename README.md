# Portable-Coronavirus-Diagnostic-Device
## Portable
Coronavirus disease 2019 (COVID-19) is an infectious disease with first symptoms similar to the flu. COVID-19 appeared first in China and quickly spreads to the rest of the world, causing the 2019-20 coronavirus pandemic. In many cases, this disease causes pneumonia. Since pulmonary infections can be observed through radiography images, this work investigates deep learning methods for automatically analyzing query chest X-ray images with the hope of bringing precision tools to health professionals towards screening the COVID-19 and diagnosing confirmed patients.
